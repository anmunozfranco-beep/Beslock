# Fingerprint Interaction

## Use when
The asset must show enrollment, unlock, retry, or troubleshooting around a fingerprint sensor.

## Prompt fragment
Adult hand using the fingerprint sensor naturally, one finger clearly aligned with the real sensor position, sharp focus on the contact zone, documentary clarity, neutral background, technically credible posture.

## Rules
- Match the exact sensor location from the product visual profile.
- Show one finger only unless the scene explicitly needs a second hand for support.
- Make the point of contact readable without covering the full device.
- Preserve hardware shape around the sensor.

## Avoid
- touching the wrong area of the product
- oversized glowing sensor effects
- impossible hand angles
- heavy motion blur
