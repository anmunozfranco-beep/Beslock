# App Pairing

## Use when
The image supports Smart Life onboarding, add-device flows, or app-based setup context.

## Prompt fragment
Adult user holding a smartphone near the installed smart lock, calm setup moment, phone and lock both visible, domestic environment, realistic materials, support-documentation tone.

## Rules
- Prefer real app captures for actual UI explanation.
- Use generated phone screens only as low-detail contextual support.
- Keep the product visually dominant enough to avoid cross-product contamination.
- Frame the scene to communicate pairing, not generic phone usage.

## Avoid
- invented readable app flows
- marketing poses or smiling lifestyle shots
- phone-only scenes with the lock barely visible
