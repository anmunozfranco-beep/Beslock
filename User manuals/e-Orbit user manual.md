# e-Orbit
## Manual de usuario

## 1. Introducción
Este manual reúne una versión estructurada del uso básico y la configuración general de la cerradura inteligente e-Orbit.

Su propósito es ayudar a entender de forma práctica:
- cómo empezar a usar la cerradura
- cómo registrar accesos
- cómo administrar usuarios
- cómo cambiar configuraciones básicas
- cómo apoyarse en la aplicación cuando corresponda

> Nota: algunos flujos y códigos detectados a partir de material previo conviene validarlos visualmente con el equipo real antes de publicación definitiva.

---

## 2. Primeros pasos
Antes de empezar, verifica lo siguiente:

- la cerradura está correctamente instalada
- la puerta abre y cierra sin forzar
- el panel responde al activarlo
- ya definiste quién será el administrador principal

Orden recomendado de configuración:
1. crear el administrador principal
2. registrar un PIN
3. registrar una huella
4. probar los accesos registrados
5. revisar configuraciones básicas
6. conectar la cerradura a la app si aplica

---

## 3. Agregar un administrador
### Qué necesitas antes
- la cerradura está encendida
- el panel responde al activarlo
- tienes acceso al menú del sistema
- ya definiste quién será el administrador principal

### Ruta de menú
**Agregar usuario → Agregar administrador**

### Paso a paso
1. activa el panel de la cerradura
2. entra al menú
3. selecciona **Agregar usuario**
4. selecciona **Agregar administrador**
5. elige el método de acceso que vas a registrar
6. completa el proceso siguiendo las indicaciones del sistema
7. guarda el registro
8. prueba inmediatamente el acceso creado

### Recomendación
Para el administrador principal, deja como mínimo un PIN, una huella y, si es posible, un método adicional de respaldo.

### Qué validar después
- el acceso funciona correctamente
- la persona puede entrar sin dificultad
- la cerradura sigue respondiendo con normalidad
- existe al menos un método de respaldo

---

## 4. Agregar un usuario
### Qué necesitas antes
- la cerradura está encendida
- ya existe al menos un administrador configurado
- sabes qué método de acceso vas a registrar

### Ruta de menú
**Agregar usuario → Agregar usuario**

### Paso a paso
1. activa el panel de la cerradura
2. entra al menú
3. selecciona **Agregar usuario**
4. elige **Agregar usuario**
5. selecciona el método de acceso
6. completa el registro
7. guarda el cambio
8. prueba el acceso

### Recomendación
Agrega un usuario a la vez y valida su acceso antes de continuar con el siguiente.

### Qué validar después
- el acceso funciona correctamente
- la persona sabe cómo usarlo
- el usuario no quedó registrado como administrador por error

---

## 5. Registrar una contraseña o PIN
### Ruta de menú
**Agregar usuario → Agregar administrador / Agregar usuario**

### Paso a paso
1. entra a **Agregar usuario**
2. elige si vas a registrar un administrador o un usuario regular
3. selecciona **Contraseña**
4. ingresa la clave
5. confírmala
6. guarda el registro
7. prueba el PIN

### Recomendación
Evita claves obvias y no compartas el PIN principal del administrador.

### Qué validar después
- la clave abre correctamente
- el usuario la recuerda
- el sistema acepta el acceso de forma consistente

---

## 6. Registrar una huella
### Qué necesitas antes
- el dedo está limpio y seco
- la cerradura responde normalmente
- el usuario puede apoyar el dedo con comodidad

### Ruta de menú
**Agregar usuario → Agregar administrador / Agregar usuario**

### Paso a paso
1. entra al menú
2. selecciona **Agregar usuario**
3. elige el tipo de usuario
4. selecciona **Huella**
5. coloca el dedo sobre el lector cuando el sistema lo indique
6. retira y vuelve a colocar el mismo dedo las veces necesarias
7. espera la confirmación
8. prueba la huella

### Recomendación
Si eres administrador, deja registrada más de una huella si el sistema lo permite.

### Qué validar después
- la lectura es estable
- el acceso funciona varias veces seguidas
- el usuario apoya el dedo con naturalidad

---

## 7. Cambiar el idioma
### Qué necesitas antes
- la cerradura está encendida
- el panel responde al activarlo
- el usuario tiene acceso al equipo

### Flujo detectado
La extracción disponible muestra ingreso con `888#` y selección de español con `3009#`. Conviene validar estos códigos visualmente con la versión real del equipo antes de publicar una versión definitiva del manual.

### Paso a paso
1. activa el panel
2. ingresa `888#`
3. entra al sistema de idioma
4. selecciona el idioma deseado
5. para español, usa `3009#` si corresponde a tu versión
6. confirma el cambio si el sistema lo solicita

### Qué validar después
- el idioma sí cambió
- el usuario entiende el menú
- el sistema sigue respondiendo normalmente

---

## 8. Conectar la cerradura a la aplicación
La app asociada para este proyecto es **Smart Life**.

### Flujo general
1. abre **Smart Life**
2. pulsa **Agregar dispositivo** o el símbolo **+**
3. entra a la categoría correspondiente
4. elige la opción de cerradura adecuada
5. sigue el flujo de vinculación

### Recomendación
Si ves varias opciones parecidas, elige la que mejor corresponda a tu modelo real.

---

## 9. Vincular por QR
### Qué necesitas antes
- la app Smart Life está instalada
- la cerradura está en modo de vinculación
- el celular tiene suficiente brillo en pantalla

### Paso a paso
1. abre **Smart Life**
2. entra a **Agregar dispositivo**
3. selecciona la categoría correspondiente
4. sigue el flujo hasta que la app muestre el código QR
5. lleva la cerradura al modo de vinculación
6. muestra el QR del celular frente al equipo
7. espera la confirmación
8. completa el proceso en la aplicación

### Recomendación
Si el QR no se detecta en el primer intento, ajusta una sola variable a la vez: brillo, distancia o iluminación.

### Qué validar después
- el dispositivo aparece en la app
- la conexión parece estable
- puedes entrar al panel del dispositivo

---

## 10. Problemas frecuentes
### La cerradura no reconoce la huella
- limpia y seca el dedo
- vuelve a apoyar la huella de forma natural
- si sigue fallando, registra nuevamente la huella
- si es administrador, deja una segunda huella de respaldo

### La cerradura no conecta a la app
- verifica que la cerradura esté encendida
- confirma que el equipo sigue en modo de vinculación
- revisa que elegiste la categoría correcta en Smart Life
- confirma el nombre y la contraseña de la red si aplica
- intenta nuevamente el proceso
