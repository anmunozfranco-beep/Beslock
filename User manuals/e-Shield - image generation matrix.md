# e-Shield
## Matriz de imágenes para DALL·E u otra IA gráfica

## 1. Objetivo del documento
Este documento organiza las imágenes necesarias para la implementación inicial del centro de ayuda y materiales de apoyo de `e-Shield`.

## 2. Imágenes prioritarias
- hero principal del producto
- producto instalado en contexto real
- interacción con panel
- uso de acceso principal
- app / vinculación
- troubleshooting básico
- descargas

## 3. Prompts maestros por categoría
### Producto hero
Cerradura inteligente moderna e-Shield instalada en una puerta residencial, fotografía realista de producto, ambiente limpio, iluminación suave, composición clara, estilo técnico y sobrio.

### Acción de uso
Persona adulta interactuando con una cerradura inteligente e-Shield, acción clara, fotografía realista, fondo discreto, estilo documental.

### App + producto
Persona usando smartphone junto a una cerradura inteligente e-Shield, escena realista de configuración o vinculación, ambiente limpio, composición didáctica.
