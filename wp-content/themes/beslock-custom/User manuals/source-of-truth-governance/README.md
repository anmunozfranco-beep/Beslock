# Source-of-Truth Governance (Layer 33)

Knowledge intake, evidence routing, operational navigation, and intake safety.

Modeling-only. Subordinate to layer 32 (multimodal evidence governance) and to
every prior governance layer. Writes no per-product artefact; mutates no runtime.

Subdirectories:

- `governance/intake-routing/` — 10-step intake pipeline + routing policy
- `governance/evidence-taxonomy/` — operator-facing format → class/scope/contract map
- `governance/trust-models/` — operator-facing trust rules anchored to layer 32
- `governance/routing-examples/` — worked 'where does this go?' examples
- `governance/intake-consultation/` — reviewer-facing consultation protocol
- `portfolio-vs-product-governance/` — scope taxonomy + folder layouts
- `operational-navigation/` — ecosystem self-orientation
- `intake-safety-governance/` — block-on-detection safety rules
- `future-copilot-intake-readiness.json` — interface contract for future Copilot
