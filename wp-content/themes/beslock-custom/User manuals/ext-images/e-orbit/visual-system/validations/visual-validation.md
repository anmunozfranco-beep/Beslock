# e-Orbit Visual Validation

## Product-specific pass checks
- The product remains a tall integrated smart lock with a strong vertical body.
- The upper front sensor or camera cluster remains present.
- The keypad, display area, and handle remain distinct zones.
- The fingerprint sensor remains on the upper exterior handle.
- Interior views preserve the screen-plus-thumbturn relationship.

## Slot cautions
- Hero and installed-context scenes must not crop out the features that distinguish e-Orbit from simpler locks.
- App scenes must not fabricate UI detail as technical truth.
- Troubleshooting scenes must show the correct handle sensor location.
