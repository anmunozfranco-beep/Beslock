# e-Orbit
## Guía de estilo visual

## 1. Objetivo
Definir criterios visuales funcionales para el centro de ayuda y el manual de e-Orbit.

---

## 2. Principios visuales
- claridad sobre decoración
- jerarquía fuerte
- escaneabilidad primero
- consistencia entre páginas
- reutilización de bloques

---

## 3. Componentes visuales sugeridos
- encabezado de página
- checklist
- caja de ruta de menú
- pasos numerados
- caja de recomendación
- caja de advertencia
- problemas frecuentes
- siguientes pasos

---

## 4. Criterios de imagen
- imágenes útiles, no decorativas
- producto claro y realista
- fondos limpios
- una acción por imagen
- evitar texto generado dentro de imagen
