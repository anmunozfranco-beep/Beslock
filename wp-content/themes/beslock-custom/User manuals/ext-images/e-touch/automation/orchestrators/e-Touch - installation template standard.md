# e-Touch
## Anexo tecnico de instalacion normalizado para Beslock

## 1. Rol del documento
Este anexo normaliza el uso del material tecnico de instalacion hoy disponible para e-Touch.

La fuente OEM corta sirve como guia ilustrada de montaje.
No sirve como plantilla 1:1 de perforacion.

## 2. Fuente de origen
- archivo fuente principal: `e-Touch_1.pdf`
- respaldo textual: `../../../../generated_manuals/e-touch/manual.md`

## 3. Posicion dentro del estandar Beslock
Este anexo tecnico acompana al:
- manual de uso
- manual de instalacion
- manual de app

Su funcion es explicar como debe leerse la guia OEM mientras no exista una plantilla de corte a escala real.

## 4. Que si manda esta fuente
- identidad de piezas del packing list
- diagrama explotado del montaje
- orden practico de instalacion
- criterio de ajuste de tornillos segun grosor de puerta
- orientacion del pestillo para interior/exterior
- energia, bateria y respaldo `micro-USB`

## 5. Que no debe hacerse con esta fuente
- no usarla como patron final de perforacion
- no reconstruir cotas definitivas desde el PNG o desde una impresion no controlada
- no presentarla como plantilla equivalente a una hoja de drilling template OEM

## 6. Regla de uso operativo
1. usar `e-Touch_1.pdf` para entender piezas, montaje y validacion inicial
2. usar el OCR solo como apoyo textual cuando la imagen no sea suficiente
3. si llega una plantilla 1:1 del proveedor, esa fuente reemplaza a esta como referencia de corte

## 7. QA minimo antes de montar
- confirmar direccion del pestillo
- confirmar grosor de puerta para escoger tornillos o conectores
- confirmar paso correcto del eje cuadrado y del cable
- no perforar una puerta nueva basandose solo en el PDF corto renderizado

## 8. Entregables Beslock asociados
- PDF tecnico: `e-Touch_1.pdf`
- render principal: `assets/e-touch/pdf-renders/e-touch-1/page-01.png`
- render de soporte y reset: `assets/e-touch/pdf-renders/e-touch-1/page-02.png`
- auditoria de fuente suplementaria: `e-Touch - supplemental source review.md`

## 9. Decision editorial
En e-Touch este documento debe marcarse como `guide-only` dentro de la matriz documental.

No reemplaza al manual de instalacion.
No reemplaza al manual de app.
No equivale a una plantilla 1:1.
