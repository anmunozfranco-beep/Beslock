# e-Touch Visual Validation

## Product-specific pass checks
- The product remains a handle-led lever set with round roses.
- The numeric markings remain on the exterior handle itself.
- The fingerprint sensor stays near the exterior handle base.
- No long escutcheon plate, screen, camera, or knob traits appear.
- The interior handle remains simpler than the exterior side.

## Slot cautions
- PIN scenes must use the handle numeral area, not an invented keypad slab.
- Fingerprint scenes must target the handle-base sensor.
- App scenes must preserve the round-rose geometry in frame.
