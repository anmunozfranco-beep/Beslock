# e-Shield Visual Profile

## Reference anchor
- Current canonical reference image: `../e-Shield.png`
- The current anchor is a transparent isolated cutout; treat the alpha silhouette as the product boundary for new compositions.
- Product family: split smart rim lock with exterior keypad slab and separate interior lock box

## Canonical identity
- exterior side is a tall, narrow, rounded-rectangle slab with glossy black face
- exterior face includes a circular fingerprint sensor with a blue accent ring near the top
- numeric keypad sits below the sensor on the exterior slab
- interior side is a separate horizontal rectangular lock housing
- no lever handle and no knob architecture

## Protected geometry
- keep the split-body architecture between exterior slab and interior box
- keep the exterior slab vertical and the interior housing horizontal
- keep the fingerprint sensor near the top of the exterior face
- keep the keypad below the sensor on the same slab
- preserve the lack of a lever or knob

## Forbidden variations
- no lever conversion
- no round knob conversion
- no long escutcheon plate body
- no camera or face-recognition screen
- no merging of the interior box into the exterior slab

## Interaction zones
- exterior top fingerprint sensor
- exterior keypad area
- interior button or control point on the lock box when relevant
- latch or bolt area only for installation or mechanical troubleshooting content

## Supplemental OEM geometry
- supplementary OEM quick sheet labels the hardware family as NX5 / NX5-L
- front body reference size from the OEM sheet: 145 x 58 x 50 mm
- rear box reference size from the OEM sheet: 130 x 110 x 50 mm
- exterior body is a glossy black vertical slab with a circular blue-ring fingerprint sensor above a 3x4 keypad including `*`, `0`, and `#`
- interior body is a separate horizontal lock box with a single round control button on the face
- drilling template companion uses a 60 mm edge setback and a centered main bore layout; use the printed template as the authority for exact hole placement

## Source caution
- `e-Shield_1.pdf` is useful for geometry, physical proportions, and hardware specs, but its raw menu text contains unvalidated codes and contradictory user-capacity counts
- `e-Shied_2.pdf` is a drilling template whose page title says `TOCK XT - TWIN`; keep it as a companion installation source and validate the SKU mapping before turning that label into a customer-facing claim

## Lighting and realism
Use neutral lighting that separates the glossy exterior slab from the bulkier interior box.
Do not stylize e-Shield into a generic deadbolt or lever lock.
