# Lighting Standard

## Purpose
Keep visual output technically legible and consistent across products.

## Prompt fragment
Neutral soft lighting, controlled reflections, clean exposure, subtle shadows, realistic black finishes, no harsh spotlighting, no neon glow, no heavy color cast.

## Required cues
- preserve surface boundaries on black hardware
- keep fingerprint rings, keypad marks, cameras, and handles readable
- prefer daylight-balanced or soft studio-neutral light
- reflections should describe geometry, not hide it

## Avoid
- deep contrast that erases black surfaces
- theatrical rim lights
- colored ambient lights
- glossy reflections that fake materials or distort the panel layout
