# Documentary Style

## Purpose
This module defines the baseline visual tone for Beslock documentation imagery.

## Prompt fragment
Technical documentary photography, clean composition, neutral environment, realistic product materials, instructional clarity over aesthetics, non-commercial mood, high legibility, believable domestic or installation context.

## Required cues
- calm and sober visual language
- one clear action or state per image
- product remains the main subject
- environment supports the task without stealing focus
- realism should feel useful, not cinematic

## Avoid
- advertising glamour
- dramatic storytelling
- trendy lifestyle styling
- exaggerated hero lighting
- futuristic or speculative design language
