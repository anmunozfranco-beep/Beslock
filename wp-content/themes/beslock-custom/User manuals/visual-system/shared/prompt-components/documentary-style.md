# Documentary Style

## Purpose
This module defines the baseline visual tone for Beslock documentation imagery.

## Prompt fragment
Technical manual illustration, clean schematic or hybrid composition, neutral environment, geometry-preserving product shell, instructional clarity over atmosphere, non-commercial mood, high legibility, restrained contextual support.

## Required cues
- calm and sober visual language
- one clear action or state per image
- product remains the main subject
- environment supports the task without stealing focus
- schematic or hybrid treatment should feel useful, not decorative
- no full human figures; use hand silhouettes only when required by the step

## Avoid
- advertising glamour
- dramatic storytelling
- trendy lifestyle styling
- exaggerated hero lighting
- futuristic or speculative design language
- photorealistic people in domestic scenes
