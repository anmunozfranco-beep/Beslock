# Review Checklist

## Pass conditions
- [ ] The product is still unmistakably the intended Beslock model.
- [ ] Protected geometry matches the visual profile.
- [ ] The interaction zone is visible and correctly used.
- [ ] The chosen realism class fits the instructional goal.
- [ ] Lighting preserves black-surface readability.
- [ ] Hands, reflections, and materials look credible.
- [ ] No generated text is carrying technical meaning.
- [ ] The asset can be assigned to a stable filename and manual slot.

## Reject when
- [ ] another product's geometry leaks in
- [ ] the lock family changes
- [ ] the key interaction area is hidden or wrong
- [ ] the image is visually attractive but not instructional
