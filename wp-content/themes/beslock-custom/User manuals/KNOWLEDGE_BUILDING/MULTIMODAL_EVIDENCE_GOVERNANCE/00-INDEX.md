# Multimodal Evidence Governance — Constitutional Index (Layer 32)

1. Evidence is the foundation; every publication step is anchored in a classified, trust-weighted, lineage-traceable artefact.
2. Ingestion is contract-bound; every format has explicit extraction, lineage, provenance, and reviewer expectations.
3. Trust is hierarchical and propagates downstream.
4. Lineage is append-only; prior events are immutable.
5. Cross-modal grounding is reviewer-curated and deterministic; no ML in this layer.
6. Lifecycle is explicit; only `promoted` artefacts can publish.
7. Conflicts are detected, never silently resolved; unresolvable cases escalate.
