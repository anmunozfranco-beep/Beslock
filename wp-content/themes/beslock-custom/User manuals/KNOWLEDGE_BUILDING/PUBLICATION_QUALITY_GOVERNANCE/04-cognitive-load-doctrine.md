# Cognitive-load doctrine

- Progressive disclosure: present what the current decision needs, defer the rest.
- Chunk size: ≤ 7 steps per atomic procedure block.
- One verb, one observable outcome, one decision boundary per step.
- Repetition is a signal of a missing shared block, not a feature.
- Ambiguity (vague pronouns, vague quantifiers, passive voice) is a defect.
