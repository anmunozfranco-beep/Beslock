observed numeric flow

Menu path: observed numeric flow

Steps:

  1. Wake the panel.
  2. Enter 888#.
  3. Enter the language system.
  4. Select the target language.
  5. Use 3009# for Spanish only if the commercial build confirms it.
  6. Confirm the change when prompted.

Validation checks:

  • The language actually changes.
  • The user understands the menu.
  • The lock still responds normally.

Smart Life -> Add device -> Camera&Lock -> Video Lock (Wi-Fi)

Menu path: Smart Life -> Add device -> Camera&Lock -> Video Lock (Wi-Fi)

Steps:

  1. Open Smart Life.
  2. Start add-device flow.
  3. Choose the lock category.
  4. Enter Wi-Fi details.
  5. Put the lock in network mode.
  6. Show the QR from the phone to the lock.
  7. Wait for confirmation.
  8. Complete the app flow.

Validation checks:

  • The device appears in the app.
  • Connection appears stable.
  • The device panel can be opened in the app.

Menu path: Agregar usuario -> Agregar administrador or Agregar usuario

  2. Choose Agregar usuario.
  3. Choose the user type.

  7. Wait for confirmation.
