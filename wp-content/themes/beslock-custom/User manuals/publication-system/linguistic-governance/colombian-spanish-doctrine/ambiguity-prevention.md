# Ambiguity prevention

Forbidden constructs in instructional text:

- Vague pronouns without antecedent (`hágalo`, `tóquelo`).
- Vague quantifiers in steps (`unos segundos`, `varias veces`, `algunos pasos`).
- Ambiguous time markers (`luego`, `después` without explicit ordering).
- Conditional steps without a resolved branch (`si es necesario`).
- Passive voice with deleted agent (`se debe configurar`).
- Redundant fillers (`de manera adecuada`, `correctamente`, `según corresponda`).

Required constructs:

- Sujeto + verbo + objeto + observación esperada.
- Ramas explícitas con resolución completa.
- Tiempos cuantificados (`espere 10 segundos`, no `espere un momento`).
