# Readability rules (Colombian operational Spanish)

- Sentence length: 6–18 words per instructional sentence; hard cap 24.
- One verb of action per step. Compound steps joined by `y`, `luego`, `después` are   forbidden when each clause is a separate observable action.
- Prefer active voice (`Pulse el botón`) over passive (`El botón debe ser pulsado`).
- Prefer concrete objects (`la cerradura`, `la app`, `el código`) over pronouns.
- Conditional branches must be explicit: `Si X, realice A; de lo contrario, realice B.`
- Tooling, menu paths and UI labels are wrapped in `«…»` and preserve original casing.
- Numbers in steps are written as digits (`5`, `15 s`), not words.
